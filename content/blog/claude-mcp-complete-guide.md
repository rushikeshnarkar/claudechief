# The Complete Guide to Claude MCPs in 2025

Model Context Protocol (MCP) is changing how we interact with AI. Learn how to extend Claude with powerful integrations.

## What is MCP?

MCP (Model Context Protocol) is an open protocol that enables AI models to connect with external tools and services. Think of it as a universal adapter for AI.

## Why MCPs Matter

Without MCPs, Claude is powerful but isolated. MCPs let Claude:
- Access your filesystem
- Send messages on Slack
- Search the web
- Connect to databases
- And much more

## Getting Started with MCPs

1. Visit our [MCPs directory](/mcps)
2. Choose the integration you need
3. Follow the installation instructions
4. Start using it in Claude Code

## Popular MCPs to Try

### Filesystem MCP
Access and manipulate files directly from Claude.

### Slack MCP
Send messages and manage channels without leaving Claude.

### GitHub MCP
Manage repositories, issues, and pull requests.
